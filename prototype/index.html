UI 原型（V2）

V2 仅保留一条全链路 HTML：微信登录（对齐 pages/login）→ 暖纸 / 青黛页内切换 → 垂直世系族谱树 → 成员详情与管理表单 → 家族文化（捐献榜单 · 家族活动 · 绑定谱员）→ 创建家族 / 添加成员。路由与栈式返回对齐 miniprogram/app.json；样式上 shared-styles.css 只管帧与变量，页面与组件见 v2/app.css。
